🚀 GORZEN System

GORZEN — это экосистема для проведения интерактивных викторин (квизов) в локальной сети. Идеально подходит для школ, офисов или дружеских вечеринок.
🛠 Состав проекта:
1. GORZEN Client/Server (GorZEN.py)

Основное приложение, которое может работать в двух режимах:

    Режим Админа: Создает сервер, управляет ходом игры, переключает вопросы и видит таблицу лидеров.

    Режим Игрока: Автоматически находит сервер в локальной сети через UDP-протокол, подключается и отвечает на вопросы.

Особенности:

    Автоматический поиск серверов (Zero-config).

    Синхронизация таймера между всеми участниками.

    Поддержка русского и армянского языков.

2. GORZEN Editor (GorZEN Editor.py)

Профессиональный инструмент для создания контента.

    Интерфейс: Удобное окно на PyQt6 для написания вопросов.

    Настройка времени: Можно выставлять индивидуальный лимит времени для каждого слайда.

    Экспорт: Сохраняет все вопросы в формат .json, который легко читается основным приложением.

🚀 Как запустить / How to Run

    Установите зависимости:
    Bash

    pip install PyQt6

    Создайте квиз: Запустите GorZEN Editor.py, напишите вопросы и сохраните файл как quiz.json.

    Запустите сервер: Запустите GorZEN.py на главном ПК и выберите "Создать сервер".

    Подключитесь: Запустите GorZEN.py на других ПК в той же сети — они увидят сервер в списке!
